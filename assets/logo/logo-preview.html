Usage Guidelines

Header Navigation

Use the full logo in website headers. Minimum clear space: 20px around logo.

Footer

Use either full logo or icon logo in website footer. Maintain brand consistency.

Favicon

Use the icon logo scaled to 16x16 or 32x32 pixels for browser tabs.

Social Media

Use icon logo for profile pictures. Use full logo for cover images when space allows.

Print Materials

Use SVG for vector printing or high-resolution PNG for raster printing.

Dark Backgrounds

Logo is optimized for dark themes (#0C0F14). Maintain sufficient contrast.

PNG Generation Instructions

To generate PNG files from SVG:

  1. Open SVG files in any vector editor (Inkscape, Adobe Illustrator)
  2. Export as PNG with transparent background
  3. Recommended sizes:
    • Full logo: 800x240px (for web), 2000x600px (for print)
    • Icon: 512x512px (social media), 180x180px (Apple Touch Icon)
    • Favicon: 32x32px, 16x16px
  4. Save files in the assets/logo/ directory
